When you start `claw` without a subcommand it opens an interactive REPL. Slash commands control the session, inspect state, and trigger built-in workflows. Tab completion expands slash commands, model aliases, permission modes, and recent session IDs as you type.

```bash
claw          # open the REPL
/help         # list all slash commands
```

    | Command | Description |
    |---------|-------------|
    | `/plugin list` | List all installed plugins and their enabled/disabled state. |
    | `/plugin install <path>` | Install a plugin from a local path or URL. |
    | `/plugin enable <name>` | Enable a previously disabled plugin. |
    | `/plugin disable <name>` | Disable a plugin without uninstalling it. |

    /plugin install ./my-plugin

    /plugin disable my-plugin

## Key command details

### /plugin

Aliases: `/plugins`, `/marketplace`

```text
/plugin list
/plugin install <path>
/plugin enable <name>
/plugin disable <name>
/plugin uninstall <id>
/plugin update <id>
```

### /skills

```text
/skills list
/skills install <path>
/skills help
```

| Sub-command | Description |
|-------------|-------------|
| `list` | Print all skills discovered in `.claw/skills/` and user-level skill directories. |
| `install <path>` | Install a skill from a local path. |
| `help` | Print usage information for the `/skills` command. |

### /subagent

```text
/subagent list
/subagent steer <target> <message>
/subagent kill <id>
```

| Sub-command | Description |
|-------------|-------------|
| `list` | List all active sub-agents and their current status. |
| `steer <target> <message>` | Send a steering message to a running sub-agent. |
| `kill <id>` | Terminate a sub-agent by ID. |

### /mcp

```text
/mcp list
/mcp show <server>
/mcp help
```

| Sub-command | Description |
|-------------|-------------|
| `list` | List all configured MCP servers and their lifecycle state. |
| `show <server>` | Print configuration and tool manifest for a specific MCP server. |
| `help` | Print usage information for the `/mcp` command. |

<Note>
  The REPL supports tab completion for all slash commands. Press `Tab` after typing `/` to see the full list, or after a command name to expand sub-commands and arguments.
</Note>
