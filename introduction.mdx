Claw Code is the open-source Rust implementation of the `claw` CLI agent harness. It gives you an interactive REPL, one-shot prompt execution, session persistence, a permission system, a plugin ecosystem, and MCP server management — all wired to multiple AI providers through a single binary.

## What Claw Code is

Claw Code is a **build-from-source** CLI. There is no hosted service and no npm/PyPI package to install. You clone the repo, run `cargo build`, and get the `claw` binary. The binary talks directly to the AI provider of your choice using your own credentials.

The project is designed to be **clawable** — meaning it is operated as much by autonomous coding agents as by humans. The same binary that a developer uses interactively in a terminal is the one that an agent orchestrator drives through sessions, hooks, and structured JSON output.

## Key capabilities

<Steps>
  <Step title="Build and verify">
    Clone the repo, run `cargo build --workspace` from `rust/`, and verify your setup with `claw doctor`.
  </Step>
  <Step title="Set credentials">
    Export `ANTHROPIC_API_KEY`, `OPENAI_API_KEY`, or another provider credential. Claw routes to the right backend automatically based on the model name prefix.
  </Step>
  <Step title="Run a prompt or open the REPL">
    Use `claw prompt "your task"` for a one-shot run, or just `claw` to open the interactive REPL with tab-completion and slash commands.
  </Step>
  <Step title="Extend with plugins and MCP">
    Install plugins with `/plugin install`, connect MCP servers via `.claw/settings.json`, and manage sessions with `/session` and `--resume`.
  </Step>
</Steps>

<Note>
  `cargo install claw-code` installs a deprecated stub — **not** this project. Build from source or use `cargo install agent-code` for the upstream binary. See the [installation guide](/installation) for details.
</Note>
